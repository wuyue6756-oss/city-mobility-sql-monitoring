## 1. dim_zone（区域维表）
用于存储区域编号与区域名称的映射关系。

字段：
- LocationID：区域编号
- Borough：行政区
- Zone：区域名称
- service_zone：服务区域类型

## 2. fact_trip（订单事实表）
用于存储出租车订单明细数据。

核心字段：
- tpep_pickup_datetime：上车时间
- tpep_dropoff_datetime：下车时间
- passenger_count：乘客数
- trip_distance：行程距离
- PULocationID：上车区域编号
- DOLocationID：下车区域编号
- fare_amount：基础车费
- tip_amount：小费
- total_amount：订单总金额
- payment_type：支付方式

## 3. vw_trip_clean（清洗视图）
用于过滤掉明显异常数据，如：
- trip_distance <= 0
- total_amount <= 0
- 上下车区域为空
- 时间字段为空
