# 指标口径说明

## 1. 订单量（trip_cnt）
统计周期内满足清洗条件的订单记录数。

## 2. 总收入（total_revenue）
统计周期内 total_amount 字段求和后的结果。

## 3. 客单价（avg_order_value）
统计周期内 total_amount 的平均值。

## 4. 平均行程距离（avg_trip_distance）
统计周期内 trip_distance 的平均值。

## 5. 异常高收入日期
按天聚合后，对 total_revenue 进行排序，选取收入最高的日期作为异常观察对象。

## 6. 区域热度
按上车区域（PULocationID）聚合，统计订单量或收入，用于识别高热度区域。

## 7. 异常日期识别（z_score）
基于每日 total_revenue 计算整体均值与标准差，
使用 z-score 衡量单日收入偏离整体均值的程度，
并按 z_score 从高到低排序，识别异常程度较高的日期。
